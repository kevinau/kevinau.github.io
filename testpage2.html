Entry mode

Whether an data entry field allows data entry, is view only, or is not applicable to the form and is hidden.

There is an enumeration “EntryMode” that is used for data entry mode:

EntryMode.ENTRY
Normal data entry is allowed.
EntryMode.VIEW
The value of the data entry field is displayed but no changes can be made.
EntryMode.NA
The data entry field is not applicable in this form. For example, if a form has a checkbox for “isTaxApplicable”, a tax rate field would be not applicable if this checkbox was clear.

If an entry field is not applicable, the field and field label are not shown—they occupy no space of the form.=:
