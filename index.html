--- layout: default title: j2form - Java to form mapping ---

j2form mapps a Java object to a data entry form. It is a Java implementation of object to form mapping (ofm) which maps a programming language object (a Java object in this case) to a data entry form.

More information can be found in the documentation pages of this site.
